# Decision Records

Dokumentierte Entscheidungen mit Optionen, Begründung und Risiken.

## Namenskonvention

```
DEC-001__kurzbeschreibung.md
```

## Regeln

- Jede nicht-triviale Entscheidung (Architektur, Tooling, Strategie) bekommt ein DEC.
- Status: `proposed` → `accepted` | `rejected` | `superseded`
- Template: `docs/templates/DECISION.md`
- Nummerierung: fortlaufend (DEC-001, DEC-002, ...).
