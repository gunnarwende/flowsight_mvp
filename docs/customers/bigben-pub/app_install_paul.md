# BigBen Pub — App auf Pauls Handy installieren

**Was:** PWA (Progressive Web App) — verhält sich wie eine echte App, läuft direkt aus dem Browser. Kein App-Store, keine Updates manuell, kein Speicherplatz-Hunger.

**Wo:** `flowsight.ch/ops/app/bigben-pub`

---

## iPhone (Safari) — Schritt-für-Schritt

1. **Safari öffnen** (nicht Chrome — PWA-Install funktioniert nur in Safari auf iOS)
2. URL eintippen: `flowsight.ch/ops/app/bigben-pub`
3. Seite lädt → unten **Teilen-Symbol** ⬆️ tippen
4. Im Menü nach unten scrollen → **"Zum Home-Bildschirm hinzufügen"**
5. Name bestätigen: **"BigBen Pub"** (oder anpassen)
6. Oben rechts **"Hinzufügen"** tippen
7. App-Icon erscheint auf Pauls Home-Screen
8. Icon antippen → App öffnet sich im Vollbild (kein Browser-Frame)
9. **Login:** Email eintippen → 6-stelliger Code aus Email → fertig

---

## Android (Chrome) — Schritt-für-Schritt

1. **Chrome öffnen**
2. URL eintippen: `flowsight.ch/ops/app/bigben-pub`
3. Seite lädt → oben rechts **Drei-Punkte-Menü** tippen
4. **"App installieren"** oder **"Zum Startbildschirm hinzufügen"**
5. Name bestätigen → **"Hinzufügen"**
6. App-Icon erscheint auf Pauls Home-Screen
7. Icon antippen → App öffnet
8. **Login:** Email eintippen → 6-stelliger Code → fertig

---

## Erste Schritte in der App (für Paul)

### Push-Benachrichtigungen einschalten

Beim ersten Öffnen fragt die App: "Möchtest du Benachrichtigungen erhalten?"
- **JA** tippen → Paul bekommt Push bei jeder neuen Reservierung

Falls Paul "Nein" tippt: in Settings → Push-Notifications → erneut aktivieren.

### Was Paul auf einen Blick sieht

**Dashboard (Hauptansicht):**
- Wie viele Reservierungen heute
- Wie viele für die nächsten 7 Tage
- Welche Events anstehen
- Aktuelle Walk-in-Zahlen

**6 Cards (oben):**
1. **Reservations** — eingehende Anfragen, bestätigen/ablehnen
2. **Guest Watch** — Liste der Gäste die nicht erschienen sind (Yellow/Red Card)
3. **Events** — wöchentliche Events (Karaoke Mi, Quiz Fr, ...)
4. **Sports** — Spiele/Übertragungen
5. **Website** — Direktlink zu seiner Pub-Website
6. **Bookings Analytics** — Walk-in/Online/Phone-Statistik

## Wenn was nicht klappt

### "App lässt sich nicht installieren"
- iPhone: nur in Safari, nicht in Chrome
- Android: in Chrome, nicht in Firefox
- Alternative: URL als Bookmark setzen — funktioniert auch

### "Login schlägt fehl — kein Code in Email"
- Spam-Ordner prüfen
- Email-Adresse korrekt geschrieben?
- 1 Min warten + nochmal versuchen

### "App zeigt nichts an"
- Browser-Cache leeren (Settings → Safari/Chrome → Cache löschen)
- App neu installieren

### "Push kommt nicht an"
- Settings (Handy-Settings, nicht App!) → BigBen Pub App → Notifications einschalten
- App einmal öffnen + schliessen
- iOS: Push erst ab iOS 16.4
