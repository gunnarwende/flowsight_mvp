### Klare Modul-2-Betriebe (brauchen KEINE FlowSight-Website)

**30 von 42 = 71%**

Diese Betriebe haben eine funktionale bis moderne Website. FlowSight ergaenzt ueber Voice + Wizard + Leitzentrale + Reviews + SMS. Keine Website noetig.

### Grenzfaelle (schwache Website, aber Modul 2 moeglich)

**5 von 42 = 12%**

Sanitherm, Widmer H., Schaub Sanitaer, Benzenhofer, Beeler — alle haben SSL-Probleme oder Baukasten-Sites. Fuer diese reicht `/start/[slug]` als Wizard-Einstieg. Eine FlowSight-Website waere nett, aber nicht entscheidend fuer den Sale.

### Disqualifiziert

**2 von 42 = 5%**

Scherrer (eingestellt), Fust (Enterprise-Kette).

## Die entscheidende Zahl

**Von 42 analysierten Betrieben in Kanton Zuerich sind genau 3-5 echte Modul-1-Kandidaten.**

Das sind **7-12% des Marktes.**

Von diesen 3-5:
- 2 haben KEINE Website (muessen irgendwas bekommen)
- 2 haben KAPUTTE Websites (SSL-Fehler)
- 1 hat eine VERALTETE Website (Adobe Muse)

Keiner davon ist ein "ich brauche eine high-end Website" Fall. Alle sind "ich brauche IRGENDWAS das funktioniert" Faelle.

---

## Implikation fuer die Modul-1/Modul-2-Entscheidung

### Die Daten sagen klar:

1. **71% der ICP-Betriebe haben brauchbare bis moderne Websites.** Modul 2 ist der natuerliche Fit.

2. **Die 5 Modul-1-Kandidaten brauchen keine "high-end differenzierte Website-Maschine".** Sie brauchen einen funktionalen Basisauftritt. Der Anspruch "3 verschiedene Wahrnehmungssysteme" war fuer einen Markt konzipiert, der so nicht existiert.

3. **Der echte FlowSight-Wert fuer ALLE 42 Betriebe ist identisch:** Voice + Wizard + Leitzentrale + Reviews + SMS. Die Website ist nie der Kauffaktor.

4. **Website-Pain korreliert NEGATIV mit Zahlungsbereitschaft.** Die Betriebe ohne Website (MPM, Heizen24) haben auch die geringste digitale Reife → schwierigerer Sale, laengere Einarbeitungszeit, hoehere Support-Last.

### Die richtige Produktgrenze:

```
STANDARD (100% der Betriebe):
  FlowSight = Voice + Wizard + Leitzentrale + Reviews + SMS
  Website des Betriebs bleibt wie sie ist
  Wizard-Einstieg: /start/[slug]

AUSNAHME (5-10% der Betriebe, nur wenn Website unter Minimum):
  1 sauberer FlowSight-Basisauftritt
  KEIN High-End-Versprechen
  KEIN Differenzierungsanspruch
  Pragmatisch, modern, vertrauenswuerdig
  = Das EINE Template (Legacy page.tsx) ohne Profil-System
```

---

## Empfehlung

Die Modul-1-Website-Maschine (3 Systeme, Banana, Profile, CSS-Handschriften) war fuer ein Problem gebaut, das es in dieser Groesse nicht gibt. 5 von 42 Betrieben brauchen ueberhaupt eine Website — und diese 5 brauchen keine differenzierte Maschine, sondern einen funktionalen Basisauftritt.

**Entscheidung: Modul 1 als Maschinenprodukt beenden. Website nur als Ausnahmefall fuer die wenigen Betriebe ohne jede Web-Praesenz.**
