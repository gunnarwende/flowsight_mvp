## Export in Chrome (empfohlen)

### Methode 1: DevTools Screenshot (pixelgenau)
1. Datei in Chrome öffnen (Doppelklick oder `chrome <datei>.html`)
2. `F12` → DevTools öffnen
3. `Ctrl + Shift + P` → Command Menu
4. Tippe: **"Capture full size screenshot"**
5. Enter → PNG wird heruntergeladen

### Methode 2: Device Mode (wenn Scrollbars stören)
1. Datei in Chrome öffnen
2. `F12` → DevTools → `Ctrl + Shift + M` (Device Mode toggle)
3. Oben "Responsive" → eigene Dimension eintragen:
   - Logo: `1024 × 1024`
   - Alles andere: `1920 × 1080`
4. `Ctrl + Shift + P` → "Capture screenshot" (nicht "full size"!)
5. PNG wird heruntergeladen

### Methode 3: Browser-Erweiterung
- [GoFullPage](https://chrome.google.com/webstore/detail/gofullpage) für One-Click Full Page Screenshots
