# Block E — Precision Pack (10 Minuten)

**Datei:** `block_e.wav`
**Ziel:** Jeder kritische Begriff dreifach verankert — isoliert, langsam, im Satz.
**Warum separat von Block D:** Block D = natürliche Sätze. Block E = gezielte Aussprache-Anker. Verschiedene kognitive Tasks, verschiedene Delivery.
**Ablauf pro Begriffsgruppe:** 3 Stufen:
1. **Isoliert:** Nur das Wort/den Begriff, klar und deutlich
2. **Langsam:** Gleicher Begriff, bewusst etwas langsamer und betonter
3. **Im Satz:** Ein natürlicher Satz, der den Begriff enthält

**Pause zwischen Begriffsgruppen:** 3 Sekunden.
**Pause zwischen den 3 Stufen:** 2 Sekunden.
