## Guardrails (auswendig lernen!)

| Regel | Warum |
|-------|-------|
| **Immer PLZ + Ort sagen** beim Lisa-Anruf | Ohne PLZ/Ort: Webhook rejected → kein Case im Dashboard |
| **Kein Englisch** während Lisa-Anruf (DE) | Trigger-Words ("hello", "please") feuern Language-Gate → Transfer zum INTL-Agent |
| **Max 3 Min** pro Lisa-Anruf | Agent hat 7-Min-Limit, aber Demo braucht 2 Anrufe |
| **Dry-Run-Nummer** für Tests, **Live-Nummer** für Demo | Verhindert Doppel-SMS auf dem Demo-Handy |
